<h1 style="text-align: center;">初めてのnode.jsコンソールアプリ（Typescript）</h1>

## TypeScript実行
```bash
npm run dev
```

### 継続的実行
```bash
npm run dev:watch
```

## TypeScript -> JavaScript に変換
```bash
npm run build
```

## 生成されたJavaScriptを実行
```bash
npm run start
```
### または
```bash
npm start
```

## 生成されたファイルを削除
```bash
npm run clean
```
